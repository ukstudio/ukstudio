--- layout: post title: RSpecの結果をHTMLで出力する方法 tags: - article - RSpec - Ruby status: publish type: post published: true meta: _edit_last: '2' SBM_count: 00006<>1248417401<>5<>0<>1<>0<>0 --- RSpecにはオプション-hがあるので、そこでフォーマットを指定してやればいい。以下、Railsの例。 
RAILS_ROOT$ spec spec -f h:spec/spec_report.html
これで、specディレクトリにspec_report.htmlが出力される。
